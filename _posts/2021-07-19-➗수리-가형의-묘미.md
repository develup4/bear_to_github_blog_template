## 자, 그러면 실제 수능문제로…  
   
`x²- 4 = a ( x - 2 )가 중근을 가질때 a의 값은?`  
   
수1만을 배웠을 때 풀이법은  
`( x + 2 ) ( x - 2 ) = a ( x - 2 )`  
  
양변에서 ( x - 2 )를 소거하되 x=2라는 근을 인정합니다.  
```  
x = 2, x + 2 = a  
∴ a = 4  
```  
  
혹은 판별식을 사용해서 a를 구해낼 것 입니다.   
  
  
## 근데 얼마전에 미분을 배웠죠?  
a가 x = 2에서의 접선의 기울기로군요.  
`y’ = 2x -> a = 4`  
   
전자가 더 편하다고요? 네, 더 익숙합니다.
